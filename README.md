# Instacart Reorder Behavior Dashboard

## Overview
This project analyzes customer reorder behavior using the Instacart dataset, focusing on weekly (7-day) and bi-weekly (14-day) shopping patterns.

## Features
- Reorder rate comparison (Weekly vs Bi-Weekly)
- Top 5 most reordered products
- Distribution of days between orders
- Heatmap of order activity (day of week vs hour)

## How to Run

1. Clone the repository:
   git clone <repo-url>

2. Install dependencies:
   pip install -r requirements.txt

3. Run the dashboard:
   python app.py

4. Open in browser:
   http://127.0.0.1:8050/

## Dataset
The dataset is based on the Instacart Market Basket Analysis dataset from Kaggle.
Due to size constraints, the dataset is not included in this repository.
